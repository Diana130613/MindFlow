# РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

## MindFlow — Мобильное приложение для ментального здоровья

**Версия:** 1.0  
**Платформа:** Android 5.0 и выше

---

## 1. Начало работы

### 1.1. Установка

1. Загрузите APK-файл `mindflow-debug.apk`
2. На Android-устройстве перейдите в Настройки → Безопасность → Разрешить установку из неизвестных источников
3. Откройте скачанный APK и нажмите «Установить»
4. После установки найдите приложение MindFlow в меню приложений

### 1.2. Регистрация

1. Запустите приложение
2. Нажмите «Нет аккаунта? Зарегистрироваться»
3. Введите:
   - **Email** — действующий адрес электронной почты
   - **Пароль** — минимум 8 символов
   - **Имя** — ваше имя в приложении
4. Нажмите «Зарегистрироваться»

### 1.3. Вход в систему

1. Введите email и пароль
2. Нажмите «Войти»
3. Приложение запомнит сессию — повторный вход потребуется через 7 дней

## 2. Основные функции

### 2.1. Главный экран (Home)

На главном экране вы увидите:
- **Приветствие** с вашим именем
- **Серия (Streak)** — сколько дней подряд вы практикуете
- **Рекомендуемые медитации** — на основе вашего настроения
- **Быстрая запись настроения** — 5 кнопок-эмодзи

### 2.2. Медитации

**Просмотр списка:**
1. Перейдите во вкладку «Медитации» (нижняя навигация)
2. Используйте вкладки категорий: Сон, Тревога, Фокус, Стресс, Осознанность
3. Или введите запрос в поисковой строке

**Запуск медитации:**
1. Нажмите на карточку медитации
2. Прочитайте описание и нажмите «Начать»
3. Следуйте инструкциям на экране
4. Нажмите «Завершить» в конце

> **Офлайн-режим:** Медитации, которые вы просматривали ранее, доступны без интернета.

### 2.3. Дневник настроения

**Запись настроения:**
1. Перейдите во вкладку «Настроение»
2. Передвиньте ползунок (1–10) или нажмите эмодзи
3. При желании добавьте текстовую заметку
4. Нажмите «Сохранить»

**Просмотр истории:**
- Под формой записи отображается история последних 30 дней
- Нажмите «Аналитика» для просмотра графика динамики

### 2.4. Профиль

В разделе «Профиль» вы найдёте:
- **Ваши данные:** имя, email
- **Статистика:** общее число сессий, суммарное время медитаций, текущая серия
- **Кнопка «Выйти»** — завершает сессию

## 3. Работа без интернета (офлайн-режим)

MindFlow поддерживает работу без подключения к интернету:

| Функция | Офлайн |
|---------|--------|
| Просмотр кэшированных медитаций | ✅ |
| Запуск медитации (кэш) | ✅ |
| Запись настроения | ✅ (синхронизируется позже) |
| Регистрация / вход | ❌ (требует интернет) |
| Получение новых медитаций | ❌ (требует интернет) |

При восстановлении соединения данные автоматически синхронизируются с сервером.

## 4. Часто задаваемые вопросы (FAQ)

**Q: Забыл пароль — что делать?**  
A: В текущей версии восстановление пароля осуществляется через администратора. Функция «Забыли пароль?» будет добавлена в версии 1.1.

**Q: Данные сохраняются на устройстве?**  
A: Да, все данные хранятся локально в зашифрованной базе данных (Room/SQLite) и синхронизируются с сервером.

**Q: Как отключить уведомления?**  
A: Перейдите в Настройки Android → Приложения → MindFlow → Уведомления.

**Q: Приложение не подключается к серверу**  
A: Проверьте подключение к интернету. Основные функции доступны офлайн. Убедитесь, что сервер запущен по адресу `http://10.0.2.2:8081` (для эмулятора).

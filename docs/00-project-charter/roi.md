## 5. Расчёт ROI

**Формула:** `ROI = (Доход - Затраты) / Затраты × 100%`

### За 3 года (накопленным итогом)

| Показатель | Сумма |
|------------|-------|
| Суммарный доход (3 года) | 523 500 + 2 617 500 + 8 550 500 = **11 691 500 руб.** |
| Суммарные затраты (3 года) | 734 100 + 700 000 + 900 000 = **2 334 100 руб.** |
| Прибыль | **9 357 400 руб.** |
| **ROI (3 года)** | **(9 357 400 / 2 334 100) × 100% ≈ 401%** |

---

## 6. Точка безубыточности

```
Постоянные затраты в месяц: 35 500 руб.
Средний доход с Premium-пользователя: 3 490 / 12 ≈ 291 руб./мес
Количество Premium-пользователей для безубыточности: 35 500 / 291 ≈ 122 чел.
При конверсии 4%: необходимо 122 / 0.04 = 3 050 MAU
```

**Точка безубыточности:** ~3 050 активных пользователей в месяц — достижимо за 6–9 месяцев после публичного запуска.

---

## 7. Вывод

Проект MindFlow демонстрирует **высокий потенциал ROI** (>400% за 3 года) при правильной монетизации и умеренных маркетинговых инвестициях. Даже консервативный сценарий выхода на безубыточность через 6–9 месяцев подтверждает экономическую целесообразность разработки.

Для учебного контекста ключевым показателем является соотношение **трудозатраты / функциональность**: 156 часов разработки обеспечили полноценный клиент-серверный продукт с 12 экранами, 13 эндпоинтами API и покрытием тестами >40%.

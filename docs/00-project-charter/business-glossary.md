# БИЗНЕС-ГЛОССАРИЙ

## Предметная область: Ментальное здоровье и медитация

| № | Термин | Определение |
|---|--------|-------------|
| 1 | **Ментальное здоровье** | Состояние психологического благополучия, при котором человек осознаёт свои возможности, может справляться со стрессом и вносить вклад в жизнь общества |
| 2 | **Медитация** | Практика тренировки внимания и осознанности с целью достижения ясности ума и эмоционального спокойствия |
| 3 | **Сессия медитации** | Завершённый сеанс медитативной практики с фиксированной продолжительностью и типом |
| 4 | **Дыхательная техника** | Структурированное упражнение по управлению дыханием (4-7-8, box breathing, диафрагмальное) |
| 5 | **Дневник настроения** | Инструмент для ежедневной фиксации эмоционального состояния пользователя |
| 6 | **Оценка настроения** | Числовое или категориальное выражение эмоционального состояния (1-10 или emoji) |
| 7 | **Стрик (серия)** | Непрерывная последовательность дней практики без пропусков |
| 8 | **Трек медитации** | Аудиозапись или структурированный сценарий медитативной практики |
| 9 | **Категория практики** | Тематическая группа медитаций (сон, тревога, фокус, стресс, осознанность) |
| 10 | **Прогресс пользователя** | Накопленная статистика практик: общее время, количество сессий, серии |
| 11 | **Офлайн-режим** | Функционирование приложения без подключения к интернету с использованием кэшированных данных |
| 12 | **Синхронизация** | Процесс передачи локально сохранённых данных на сервер при восстановлении соединения |
| 13 | **JWT-токен** | JSON Web Token - стандарт аутентификации, используемый для безопасной передачи данных |
| 14 | **Роль пользователя** | Набор прав доступа (USER - обычный пользователь, ADMIN - администратор) |
| 15 | **Аффирмация** | Короткое позитивное утверждение, отображаемое в приложении для психологической поддержки |
| 16 | **Осознанность (mindfulness)** | Практика полного присутствия в настоящем моменте без осуждения |
| 17 | **Биометрическая аутентификация** | Вход в приложение через отпечаток пальца или распознавание лица |
| 18 | **Аналитика эмоций** | Визуальное представление динамики настроения пользователя за период |
| 19 | **Напоминание о практике** | Push-уведомление, отправляемое пользователю в заданное время для формирования привычки |
| 20 | **Достижение (бейдж)** | Награда, присваиваемая пользователю за достижение определённой цели |
| 21 | **Профиль пользователя** | Совокупность персональных данных и настроек пользователя в системе |
| 22 | **REST API** | Архитектурный стиль взаимодействия клиента и сервера через HTTP-запросы |

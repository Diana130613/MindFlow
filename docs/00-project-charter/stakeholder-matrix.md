# МАТРИЦА СТЕЙКХОЛДЕРОВ

## Проект: MindFlow

| Стейкхолдер | Роль | Интересы | Влияние | Отношение | Стратегия |
|-------------|------|----------|---------|-----------|-----------|
| **Конечный пользователь** (человек, испытывающий стресс) | Основной пользователь | Простой интерфейс, эффективные медитации, офлайн-доступ | Высокое | Позитивное | Регулярные UX-тесты, учёт feedback |
| **Администратор системы** | Управление контентом и пользователями | Удобная панель управления, стабильность системы | Среднее | Нейтральное | Разработка интуитивного admin API |
| **Психолог/терапевт** | Эксперт предметной области | Корректность методик медитации | Среднее | Позитивное | Консультации при разработке контента |
| **Разработчик** (студент) | Автор проекта | Технически грамотная реализация, хорошая оценка | Высокое | Позитивное | Следование методическим указаниям |
| **Преподаватель** | Оценщик | Соответствие требованиям методички, PCMEF | Очень высокое | Нейтральное | Строгое следование требованиям |
| **Сервер (инфраструктура)** | Внешняя система | Надёжная работа, масштабируемость | Низкое | Нейтральное | Чёткая спецификация API |

# Customer Journey Map (CJM) — MindFlow

## Персона: Алина, 22 года, студентка

> Испытывает стресс перед сессией, плохо спит, хочет попробовать медитацию, но не знает с чего начать.

---

## Карта пути пользователя

| Этап | Действия пользователя | Мысли и чувства | Точки контакта | Боли | Возможности системы |
|------|----------------------|-----------------|----------------|------|---------------------|
| **1. Осознание** | Видит рекламу / советует подруга | «Интересно, поможет ли это?» «Не знаю, подойдёт ли мне» | Социальные сети, сарафанное радио | Страх «это не для меня», скептицизм | Простой онбординг, экран Welcome с мотивацией |
| **2. Скачивание** | Устанавливает приложение из магазина | «Надеюсь, это будет понятно» | Google Play Store | Долгая загрузка, сложная форма регистрации | Быстрая регистрация (имя + email + пароль), без лишних полей |
| **3. Первый запуск** | Регистрация → попадает на Home | «О, красиво! Но что делать дальше?» | Экран приветствия, Home | Непонимание с чего начать | Quick Actions: «Начать медитацию», «Записать настроение» |
| **4. Первая медитация** | Выбирает «Дыхание 4-7-8» (5 мин) | «Буду пробовать, это не страшно» | MeditationList → Detail → Session | Боится длинных медитаций | Сортировка по длительности, чёткий таймер, кнопка «Стоп» |
| **5. Запись настроения** | Ставит оценку 4, пишет «устала» | «Интересно следить за собой» | MoodDiary экран | Непонимание шкалы | Подписи к числам + emoji, текстовая подсказка |
| **6. Возврат на следующий день** | Открывает Home, видит стрик | «Уже 2 дня подряд — круто!» | Home (стрик/прогресс) | Забывает открыть приложение | (будущее: пуш-уведомления), значок прогресса |
| **7. Регулярное использование** | Смотрит аналитику за неделю | «Стало лучше! Вижу прогресс» | Analytics (график) | График неинформативен | Цветовая маркировка настроения, тренд |
| **8. Рекомендация** | Советует подруге | «Это реально помогает» | — | — | Простая ссылка на приложение |

## Ключевые моменты истины (Moments of Truth)

| # | Момент | Риск | Решение в MindFlow |
|---|--------|------|-------------------|
| 1 | **Регистрация** | Отток из-за сложной формы | Минимальная форма (3 поля), мгновенный вход |
| 2 | **Первая медитация** | Неудача → потеря интереса | Начинать с 5-минутных практик, понятный плеер |
| 3 | **Отсутствие интернета** | Приложение не работает | Офлайн-режим через Room кэш |
| 4 | **Нет прогресса** | Потеря мотивации | График настроения, визуализация динамики |

## Выводы для проектирования

1. **Онбординг** — экран Welcome объясняет ценность до регистрации
2. **Фильтр по длительности** — новичку важно видеть 5-минутные практики первыми
3. **Offline-first** — критично для студентов (плохой WiFi, метро)
4. **Emoji-шкала настроения** — снижает когнитивную нагрузку при вводе
5. **Аналитика** — главный retention-фактор; видеть прогресс = возвращаться

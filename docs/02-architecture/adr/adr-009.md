# ADR-009: Навигация Android - Navigation Compose

Статус: ✅ Принято

---

## Контекст

Приложение содержит несколько экранов с разными уровнями доступа:
- Экраны аутентификации (без токена)
- Основные экраны (с токеном)
- Вложенные экраны (детали медитации, дыхательные практики)

Необходима система навигации совместимая с Jetpack Compose
и поддерживающая Bottom Navigation Bar.

---

## Решение

Использовать **Navigation Compose** от Jetpack.

### Граф навигации

SplashScreen
│
├── (не авторизован)  
│       └── AuthGraph  
│               ├── LoginScreen  
│               └── RegisterScreen  
│
└── (авторизован)  
└── MainGraph  
├── HomeScreen          ← bottom nav  
├── MeditationGraph     ← bottom nav  
│       ├── MeditationListScreen  
│       └── MeditationDetailScreen  
│               └── BreathingScreen  
├── MoodDiaryScreen     ← bottom nav  
└── ProfileScreen       ← bottom nav  

---


### Реализация

```kotlin
// Sealed class для типобезопасных маршрутов
sealed class Screen(val route: String) {
    // Auth
    data object Splash      : Screen("splash")
    data object Login       : Screen("login")
    data object Register    : Screen("register")
    // Main
    data object Home        : Screen("home")
    data object MeditationList : Screen("meditations")
    data object MoodDiary   : Screen("mood")
    data object Profile     : Screen("profile")
    // Detail
    data class MeditationDetail(val id: Long) : Screen("meditation/{id}") {
        fun createRoute() = "meditation/$id"
    }
    data object Breathing   : Screen("breathing")
}

// Граф навигации
@Composable
fun MindFlowNavGraph(
    navController: NavHostController,
    isAuthenticated: Boolean
) {
    NavHost(
        navController = navController,
        startDestination = Screen.Splash.route
    ) {
        composable(Screen.Splash.route) {
            SplashScreen(
                onNavigate = {
                    if (isAuthenticated)
                        navController.navigate(Screen.Home.route) {
                            popUpTo(Screen.Splash.route) { inclusive = true }
                        }
                    else
                        navController.navigate(Screen.Login.route) {
                            popUpTo(Screen.Splash.route) { inclusive = true }
                        }
                }
            )
        }

        composable(Screen.Login.route) {
            LoginScreen(
                onLoginSuccess = {
                    navController.navigate(Screen.Home.route) {
                        popUpTo(Screen.Login.route) { inclusive = true }
                    }
                },
                onNavigateToRegister = {
                    navController.navigate(Screen.Register.route)
                }
            )
        }

        composable(Screen.Home.route) { HomeScreen() }

        composable(Screen.MeditationList.route) {
            MeditationListScreen(
                onMeditationClick = { id ->
                    navController.navigate(
                        Screen.MeditationDetail(id).createRoute()
                    )
                }
            )
        }

        composable(
            route = Screen.MeditationDetail(0).route,
            arguments = listOf(
                navArgument("id") { type = NavType.LongType }
            )
        ) { backStackEntry ->
            val id = backStackEntry.arguments?.getLong("id") ?: return@composable
            MeditationDetailScreen(
                meditationId = id,
                onStartBreathing = {
                    navController.navigate(Screen.Breathing.route)
                }
            )
        }

        composable(Screen.MoodDiary.route) { MoodDiaryScreen() }
        composable(Screen.Profile.route)   { ProfileScreen() }
    }
}
```

## Bottom Navigation Bar

```kotlin
// Элементы нижней навигации
sealed class BottomNavItem(
    val screen: Screen,
    val label: String,
    val icon: ImageVector
) {
    data object Home : BottomNavItem(
        Screen.Home, "Главная", Icons.Default.Home
    )
    data object Meditations : BottomNavItem(
        Screen.MeditationList, "Медитации", Icons.Default.SelfImprovement
    )
    data object Mood : BottomNavItem(
        Screen.MoodDiary, "Настроение", Icons.Default.Favorite
    )
    data object Profile : BottomNavItem(
        Screen.Profile, "Профиль", Icons.Default.Person
    )
}
```

## Обоснование

| Критерий         | Navigation Compose                       |
|------------------|------------------------------------------|
| Официальность    | Jetpack - официальная библиотека Google  |
| Совместимость    | Нативная интеграция с Compose            |
| Deep links       | Поддержка переходов из уведомлений       |
| Back stack       | Автоматическое управление стеком экранов |
| Типобезопасность | Sealed class для маршрутов               |

---

## Альтернативы

| Библиотека       | Причина отказа                                    |
|------------------|---------------------------------------------------|
| Accompanist Nav  | Устарел - влит в официальную библиотеку           |
| Voyager          | Сторонняя библиотека - риск прекращения поддержки |
| Decompose        | Избыточно сложен для данного масштаба             |
| Ручная навигация | Много шаблонного кода, нет back stack             |

---

## Последствия
### Положительные
- Единый граф навигации - вся навигация в одном месте
- Автоматический back stack 
- Deep links из push-уведомлений
## Отрицательные
- Передача сложных объектов между экранами только через id
- Sealed class маршрутов нужно обновлять при добавлении экранов

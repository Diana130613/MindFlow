# ADR-002: Выбор языка Backend — Java 17
 
Статус: ✅ Принято

---

## Контекст

Необходимо выбрать язык программирования для серверной части.
Требование курса — использование Java.
Spring Boot 3.x требует минимум Java 17.

---

## Решение

Использовать **Java 17** (LTS) с **Spring Boot 3.2**.

---

## Обоснование

| Критерий           | Значение                                      |
|--------------------|-----------------------------------------------|
| LTS поддержка      | До 2029 года                                  |
| Spring Boot 3.x    | Требует минимум Java 17                       |
| Современные фичи   | Records, sealed classes, text blocks, switch  |
| Экосистема         | Широкий набор библиотек и инструментов        |
| Требование курса   | Java обязателен по условию задания            |

### Возможности Java 17, используемые в проекте

```java
// Records для DTO
public record MoodStatsDto(
    double average,
    int total,
    int streak
) {}

// Text blocks для SQL в тестах
String query = """
    SELECT * FROM mood_entries
    WHERE user_id = :userId
    AND recorded_at > :from
    ORDER BY recorded_at DESC
    """;

// Enhanced switch
String getMoodLabel(int score) {
    return switch (score) {
        case 9, 10 -> "Отлично";
        case 7, 8  -> "Хорошо";
        case 5, 6  -> "Нормально";
        case 3, 4  -> "Плохо";
        default    -> "Очень плохо";
    };
}
```
## Альтернативы

| Язык / Версия    | Причина отказа                                              |
|------------------|-------------------------------------------------------------|
| Kotlin (Backend) | Требование курса — Java                                     |
| Java 11          | Устарел, нет современных фич, Spring Boot 3 не поддерживает |
| Java 21          | Нестабильная поддержка в некоторых CI-окружениях            |
| Scala            | Не соответствует требованиям курса                          |

## Последствия
### Положительные
- Records сокращают код DTO
- Text blocks улучшают читаемость SQL в тестах
- Поддержка актуальных версий Spring Boot и зависимостей
### Отрицательные
- Требует JDK 17+ на машине разработчика и сервере

## ВВЕДЕНИЕ

### Актуальность

По данным Всемирной организации здравоохранения, каждый четвёртый человек в мире сталкивается с расстройствами психического здоровья. В условиях современного ритма жизни стресс, тревожность и эмоциональное выгорание становятся всё более распространёнными явлениями. Существующие мобильные приложения для ментального здоровья (Headspace, Calm) имеют высокую стоимость подписки и не адаптированы для русскоязычной аудитории.

### Цель работы

Разработать мобильное приложение MindFlow для Android, предоставляющее пользователям инструменты ментального здоровья: медитации, дыхательные практики и дневник настроения с аналитикой. Система реализована с использованием архитектурного паттерна PCMEF.

### Задачи

1. Провести бизнес-анализ предметной области
2. Разработать модель требований (Use Case, Domain Model)
3. Спроектировать архитектуру по паттерну PCMEF
4. Реализовать бэкенд на Java (Spring Boot) и Android-клиент (Kotlin)
5. Обеспечить REST API (12 эндпоинтов) с JWT-аутентификацией
6. Реализовать офлайн-режим через Room Database
7. Провести модульное тестирование (JUnit, покрытие > 40%)
8. Задокументировать систему

## 3. РЕАЛИЗАЦИОННАЯ ЧАСТЬ

### 3.1. Технологический стек

| Компонент | Технология |
|-----------|-----------|
| Android | Kotlin, Jetpack Compose, Room, Retrofit, Hilt |
| Backend | Java 17, Spring Boot 3.2 |
| Database | PostgreSQL 15 |
| Security | JWT, BCrypt |

### 3.2. Структура проекта и PCMEF

Подробно: [`docs/05-implementation/implementation-report.md`](../05-implementation/implementation-report.md)

**Реализованные слои (Backend):**
- E - 5 Entity-классов с бизнес-методами
- F - 5 JPA-репозиториев
- M - 3 Service-интерфейса + 3 реализации
- C - 3 REST-контроллера (12 эндпоинтов)

### 3.3. Безопасность

- JWT access-токены (15 мин) + refresh (7 дней)
- BCrypt для паролей (strength=12)
- Spring Security с разграничением ролей (ROLE_USER, ROLE_ADMIN)

### 3.4. Рефакторинг

Подробно: [`docs/07-refactoring/refactoring-report.md`](../07-refactoring/refactoring-report.md)

Применены паттерны: **Data Mapper**, **Identity Map**, **Lazy Load**

## ЗАКЛЮЧЕНИЕ

В ходе выполнения курсового проекта разработано мобильное приложение MindFlow для Android с серверной частью на Java Spring Boot.

**Достигнутые результаты:**
- Реализован REST API с 12 эндпоинтами (≥8 по требованию)
- JWT-аутентификация с разграничением ролей
- 5 Entity-классов с бизнес-методами (≥5 по требованию)
- Архитектура PCMEF строго соблюдена
- Документирование всех этапов жизненного цикла
- Контейнеризация через Docker

**Перспективы развития:**
- Добавление аудиомедитаций (требует CDN)
- Биометрическая аутентификация (Fingerprint API)
- Push-уведомления через FCM
- Публикация в Google Play

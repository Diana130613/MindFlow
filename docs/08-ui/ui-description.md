# ОПИСАНИЕ ПОЛЬЗОВАТЕЛЬСКОГО ИНТЕРФЕЙСА

## Проект: MindFlow — Android-приложение

---

## 1. Обзор экранов (5+ экранов, требование выполнено)

| № | Экран | Маршрут навигации | Описание |
|---|-------|------------------|---------|
| 1 | Экран входа (Login) | `/login` | Форма входа и ссылка на регистрацию |
| 2 | Экран регистрации (Register) | `/register` | Создание нового аккаунта |
| 3 | Главный экран (Home) | `/home` | Приветствие, быстрый доступ к медитациям, статистика |
| 4 | Список медитаций | `/meditations` | Список с фильтрацией по категориям и поиском |
| 5 | Экран медитации (Player) | `/meditation/{id}` | Детали и запуск медитации с таймером |
| 6 | Дневник настроения | `/mood` | Запись оценки настроения + заметка |
| 7 | Аналитика настроения | `/analytics` | График динамики настроения |
| 8 | Профиль пользователя | `/profile` | Данные пользователя, прогресс, выход |

## 3. Описание экранов

### 3.1. LoginScreen

**Компоненты:**
- Логотип MindFlow
- TextField — Email
- TextField — Пароль (скрытый ввод)
- Button "Войти"
- TextButton "Нет аккаунта? Зарегистрироваться"

**Состояния:**
- `Loading` — кнопка заблокирована, CircularProgressIndicator
- `Error` — показ сообщения об ошибке (снэкбар)
- `Success` — переход на HomeScreen

---

### 3.2. HomeScreen

**Компоненты:**
- Приветственный заголовок с именем пользователя
- Карточка "Текущая серия (streak)" — количество дней подряд
- Секция "Рекомендуемые медитации" (горизонтальный список)
- Быстрая запись настроения (5 emoji-кнопок)
- Секция "Ваш прогресс": общее время медитаций, количество сессий

---

### 3.3. MeditationListScreen

**Компоненты:**
- SearchBar для поиска
- ChipRow — категории (Сон, Тревога, Фокус, Стресс, Осознанность)
- LazyColumn — список MeditationCard
- MeditationCard: изображение, название, длительность, уровень сложности

**Состояния:**
- `Loading` — скелетон-загрузчик
- `Empty` — "Медитации не найдены"
- `Success` — список медитаций
- `Offline` — отображает данные из Room-кэша с меткой

---

### 3.4. MeditationPlayerScreen

**Компоненты:**
- Фоновое изображение медитации
- Название и описание
- CircularCountdownTimer — обратный отсчёт
- Button "Начать" / "Пауза" / "Завершить"
- AnimatedProgress — прогресс-бар

**Поведение:**
- При нажатии "Начать" запускается таймер
- По завершении показывается экран поздравления
- Сессия сохраняется в Room, затем синхронизируется с сервером

---

### 3.5. MoodDiaryScreen

**Компоненты:**
- Заголовок "Как ты себя чувствуешь сегодня?"
- Slider (1–10) + числовое значение
- Row из 5 emoji (😢😕😐🙂😁) как быстрый выбор
- OutlinedTextField — текстовая заметка (опционально)
- Button "Сохранить"

---

### 3.6. AnalyticsScreen

**Компоненты:**
- LineChart — динамика настроения за 30/7 дней (переключатель)
- AverageMoodCard — средняя оценка за период
- MoodHistoryList — список последних записей

---

### 3.7. ProfileScreen

**Компоненты:**
- Avatar (инициалы пользователя)
- Имя и email
- ProgressStats: сессии / минуты / серия
- Button "Выйти"
